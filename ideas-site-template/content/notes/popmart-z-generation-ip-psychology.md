# 泡泡玛特与 Z 世代消费心理

泡泡玛特踩中的不是一个品类，而是一个**结构性机会**。

## 企业本质

泡泡玛特不是盲盒公司，是为现代人提供一个「可被占有的小他者」。

这个「小他者」的角色是：

- 承载自我投射
- 提供情感代偿
- 在思想混沌、真实关系稀缺的环境里，给一个可控的、安全的、可展示的身份标签

![泡泡玛特IP矩阵分析](/assets/popmart-z-generation.webp)

它不是帮你解决空虚，而是帮你把空虚转成一种高频的、轻量的、可被识别的情绪位移。

## IP 分类学及其精神内核：治愈与叛逆的双重代偿

泡泡玛特的 IP 矩阵并非随意的视觉堆砌，而是精准地将复杂的社会心理映射为两类核心精神载体。

| 类型 | 代表 | 心理需求 | 机制 |
|---|---|---|---|
| 治愈系 | Dimoo、小甜豆 | 补偿童年纯真 | 通过爱护弱小形象获得力量感，构建临时心理退行空间 |
| 叛逆系 | Labubu、小野 | 释放边缘自我 | 展现内心反差，突破规范，获得想象中的自由感 |

### 治愈系 IP：童年特权的「心理退行」

**丧失的补偿。** 这类 IP（天真无邪、柔软纯正）对应的是个体在成长中逐渐丧失的纯粹探索欲和无条件信任他人的能力。当成年人将 Dimoo 当作孩子去「养」时，会产生双重支撑：

- 通过保护弱小获得力量感，意识到自己已不再脆弱。
- 通过照顾娃娃弥补自己童年缺失的爱，构建临时的心理退行空间，在照料中实现自愈。

### 叛逆系 IP：边缘自我的「代理表达」

**反差的投射。** 叛逆系 IP（视觉上带有「边际感」或「不服管」的特征）对应的是被社会规范压抑的、带有反差的一面。许多用户平时表现得体面精致，但内心可能藏着「朋克」或「阴暗」的碎片。

通过与叛逆系 IP 的互动或展示，用户实际上是在呈现那部分被压抑的自我，从而获得想象中的自由感和对抗压力的代理通道。

这解释了为什么同一个用户可能同时买 Dimoo 和 Labubu：前者治愈白天的乖巧自我，后者释放夜间的压抑自我。

## 消费范式的结构性转型：从「叙事」到「投射」

这是最有价值的一组对比。IP 价值建立路径正在发生根本转向。

| 维度 | 投放型 IP（多巴胺模式） | 叙事型 IP（内啡肽模式） |
|---|---|---|
| 案例 | 泡泡玛特、玲娜贝儿 | 迪士尼、漫威、哈利·波特 |
| 路径 | 特征识别 → 需求匹配 → 自我投射 | 理解 → 沉浸 → 共鸣 → 认同 |
| 体验 | 快反馈、低门槛、图像化 | 深沉浸、高门槛、逻辑性 |
| 安全性 | 极高：无背景 = 无剧情崩塌 | 高风险：编剧失误 = 情感背叛 |
| 商业效率 | 即插即用，高频更新产品外观 | 周期长，依赖长期内容宇宙建设 |

### 叙事型 IP：故事与 IP 同体

以迪士尼、哈利·波特、漫威为代表。核心是**「故事与 IP 同体」**。用户需要经过漫长的链接链条——理解 → 沉浸 → 共鸣 → 认同。这种模式要求用户投入巨大的心智成本，像对待现实人类一样建立深层情感反馈，甚至为之流泪。

### 投射型 IP：即插即用的身份接口

以泡泡玛特、玲娜贝儿为代表。核心是**「特征识别与需求匹配」**。投射型 IP 不关心角色的前世今生，而是直接提供一个氛围感的支撑。它更像一个空容器，用户直接将自我的某种碎片或情绪投射进去，省略繁琐的故事铺垫。

这种模式省去了漫长的内容宇宙建设周期，通过高频更新产品外观，精准缝合「产品景观」与「自我定义」，极大提高了商业收割效率。

## 媒介环境与代际社会学特征

这种范式的转变并非偶然，而是时代背景与媒介技术共同塑造的结果。

### 短视频重塑的情绪回路

Z 世代受短视频训练，适应一种回路极短的情绪刺激模式：三到五秒内必须提供刺激。这导致他们更适应「图像化、低门槛、快反馈」的多巴胺模式，而非沉重、耗时的内啡肽式深层感动。

现代受众有时并不想要「真正的感动」，因为它意味着沉重的情感负荷、逻辑理解和伦理判断。相比之下，门槛更低、更安全、带一点「糖精味」的视觉反馈更符合快节奏生活的心理避险需求。

### 图像化社交与符号系统

投射型 IP 的使用类似于年轻人「互扔表情包」的社交仪式。用户通过展示特定的 IP 符号，无需任何逻辑解释，即可快速识别出彼此是否处于同一语境，确认对方是否属于「合得来」的群体。这大大降低了社交的压力和成本。

### 思想混乱期的身份锚点

在各种观点（主义、人设、社会期望）混杂冲突的「思想大乱斗」时代，现代人很难看清「我是谁」。消费主义此时通过产品景观直接提供身份标签。通过消费特定的投射型 IP，用户能获得短暂的「完整感」和身份确认。

## 收藏性消费的未来

未来 5–10 年收藏性消费持续走高的原因：

1. 功能去中心化：用户追求「让我高兴」，而非物理实用性。
2. 社交切口：手办、卡牌、玩偶为内向个体提供低压力的线下连接点。
3. 资产与叙事：收藏品同时是投资品和个人史记录。

## 结论

泡泡玛特的护城河不在供应链，不在 IP 设计，而在商业效率：

> 不追求彻底解决空虚，追求高频情感位移。

它把「景观」和「自我」缝合在一起，让用户每次购买都是一次轻量级的身份确认。这不是漫威式的深度信仰，而是每天一次的微型仪式。它不试图解决人的根本性孤独，而是通过高效的情感位移，满足了原子化社会中个体对精神按摩的高频需求。

## {mine}

这篇内容对 Studio 的启发是直接的。

Studio 做 fantasy world，本质上也可以走两条路：

- **叙事型 Studio**：先讲故事、搭世界观、建角色关系，用户沉浸后才能认同。类似迪士尼 / 漫威路线。
- **投射型 Studio**：先给用户一个角色身份、一个标签、一种氛围，用户直接挂上去，再慢慢展开世界。类似泡泡玛特路线。

结合之前 GTM 讨论的 Cohort First vs Studio First：

- Cohort First 更偏「投射型」：先给一个可玩的故事片段，用户立刻拥有一个角色体验。
- Studio First 更偏「叙事型」：先展示完整世界体系，用户理解后再进入。

两条路的共同资产都是 Living World Page。  
但入口的心理学逻辑完全不同：一个是「我想成为这个角色」，一个是「我理解这个世界」。

投射型更适合拉新，叙事型更适合留存。  
同一个产品需要两条路径并存。

另外，「互扔表情包」这个比喻对 Studio 的角色系统很有启发——也许不需要每个角色都有完整背景故事。可以有一些「空容器型角色」，用户直接把自己投射进去，快速建立情感连接，然后随着使用深入再慢慢展开。
