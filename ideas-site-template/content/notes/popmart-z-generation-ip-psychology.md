- 泡泡玛特的本质不是盲盒，是一个「可被占有的小他者」，承载自我投射和自我表达。
- 治愈系 IP（Dimoo、小甜豆）补偿童年纯真；叛逆系 IP（Labubu、小野）释放被社会压抑的边缘自我。
- IP 消费分两种：多巴胺模式（投射型）与内啡肽模式（叙事型），泡泡玛特是前者。
- 投射型 IP：特征识别 → 需求匹配 → 自我投射，极安全、极高商业效率、无剧情崩塌风险。
- 叙事型 IP：理解 → 沉浸 → 共鸣 → 认同，周期长、风险高，但情感更深。
- Z 世代受短视频训练，更适应短回路、快反馈的情绪模式。
- 收藏性消费将持续走高，因为它同时满足：情绪价值、低压社交、资产叙事三重需求。
- 泡泡玛特的护城河是「不追求彻底解决空虚，而追求高频情感位移」。

## {full}

# 泡泡玛特与 Z 世代 IP 消费范式

泡泡玛特踩中的不是一个品类，而是一个**结构性机会**。

## 企业本质

泡泡玛特不是盲盒公司，是为现代人提供一个「可被占有的小他者」。

这个「小他者」的角色是：

- 承载自我投射
- 提供情感代偿
- 在思想混沌、真实关系稀缺的环境里，给一个可控的、安全的、可展示的身份标签

它不是帮你解决空虚，而是帮你把空虚转成一种高频的、轻量的、可被识别的情绪位移。

## IP 分类逻辑

泡泡玛特的 IP 矩阵按心理需求分两类，不是按题材分：

| 类型 | 代表 | 心理需求 | 机制 |
|---|---|---|---|
| 治愈系 | Dimoo、小甜豆 | 补偿童年纯真 | 通过爱护弱小形象获得力量感，构建临时心理退行空间 |
| 叛逆系 | Labubu、小野 | 释放边缘自我 | 展现内心反差，突破规范，获得想象中的自由感 |

这解释了为什么同一个用户可能同时买 Dimoo 和 Labubu：前者治愈白天的乖巧自我，后者释放夜间的压抑自我。

## 核心对比：投射型 vs 叙事型

这是最有价值的一组对比。

| 维度 | 投射型 IP（多巴胺） | 叙事型 IP（内啡肽） |
|---|---|---|
| 案例 | 泡泡玛特、玲娜贝儿 | 迪士尼、漫威、孙悟空 |
| 路径 | 特征识别 → 需求匹配 → 自我投射 | 理解 → 沉浸 → 共鸣 → 认同 |
| 体验 | 快反馈、低门槛、图像化 | 深沉浸、高门槛、逻辑性 |
| 安全性 | 极高：无背景 = 无剧情崩塌 | 高风险：编剧失误 = 情感背叛 |
| 商业效率 | 极高：「即插即用」身份接口 | 周期长：依赖长期内容产出 |

泡泡玛特选了投射型这条路，本质上是选择了一个**绕过内容的高效身份引擎**。

不给你讲故事，给你一个能立即挂在身上的标签。

## 媒介环境匹配

Z 世代被短视频训练出来的特点是：

- 短回路
- 快反馈
- 图像化社交
- 低逻辑成本的情绪接驳

投射型 IP 正好匹配这套行为模式：

> 一个形象 = 一个表情包 = 一个社交语境 = 一次快速身份确认

不需要解释你是谁，你买了什么就说出了你是谁。

## 收藏性消费的未来

未来 5–10 年收藏性消费持续走高的原因：

1. 功能去中心化：用户追求「让我高兴」，而非物理实用性。
2. 社交切口：手办、卡牌、玩偶为内向个体提供低压力的线下连接点。
3. 资产与叙事：收藏品同时是投资品和个人史记录。

这一点对 Studio 世界观方向有直接借鉴意义——用户收藏的不是物品，是一段可被外化、可被展示的自我叙事。

## 结论

泡泡玛特的护城河不在供应链，不在 IP 设计，而在商业效率：

> 不追求彻底解决空虚，追求高频情感位移。

它把「景观」和「自我」缝合在一起，让用户每次购买都是一次轻量级的身份确认。这不是漫威式的深度信仰，而是每天一次的微型仪式。

## {mine}

这篇内容对 Studio 的启发是直接的。

Studio 做 fantasy world，本质上也可以走两条路：

- **叙事型 Studio**：先讲故事、搭世界观、建角色关系，用户沉浸后才能认同。类似迪士尼 / 漫威路线。
- **投射型 Studio**：先给用户一个角色身份、一个标签、一种氛围，用户直接挂上去，再慢慢展开世界。类似泡泡玛特路线。

结合之前 GTM 讨论的 Cohort First vs Studio First：

- Cohort First 更偏「投射型」：先给一个可玩的故事片段，用户立刻拥有一个角色体验。
- Studio First 更偏「叙事型」：先展示完整世界体系，用户理解后再进入。

两条路的共同资产都是 Living World Page。  
但入口的心理学逻辑完全不同：一个是「我想成为这个角色」，一个是「我理解这个世界」。

投射型更适合拉新，叙事型更适合留存。  
同一个产品需要两条路径并存。
